# Hansolo

Cli tool to automate berkshelf and chef-solo deployment

## Installation

Add this line to your application's Gemfile:

    gem 'hansolo'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hansolo

## Usage

See the binary:

    $ hansolo -h
